# CSD Statistics Experiment

## Goal

Quantify the prevalence of **crystallographic disorder** and **missing/incomplete
hydrogen atoms** in the Cambridge Structural Database (CSD), demonstrating the
practical necessity of MolCrysKit's preprocessing pipeline.

This analysis supports **Figure S1** in the manuscript.

## Environment

| Item | Value |
|------|-------|
| CSD Python API | ccdc 3.6.1 |
| CSD database version | 2024 (1,413,222 entries) |

**Note:** The CSD Python API requires an academic license from CCDC.
It cannot be installed via `pip`; it ships with its own conda environment.

## Sampling Strategy

- **Population**: All organic 3D structures in CSD
- **Filter criteria**: `has_3d_structure == True`, contains carbon, all elements within {H, C, N, O, F, P, S, Cl, Br, I, B, Si, Se, As, Te}
- **Sample size**: N = 50,000
- **Random seed**: 42 (reproducible)

## Classification Scheme

| Category | Definition |
|----------|-----------|
| **A: Crystallographic disorder** | `entry.has_disorder == True` |
| **B: Missing / incomplete H** | `hydrogen_treatment` ∈ {Unknown, Missing, Omitted} OR no H atoms present |
| **A ∪ B: Any preprocessing** | Union of A and B |

## Results (n = 50,000)

| Metric | Count | Percentage |
|--------|-------|-----------|
| Crystallographic disorder | 9,601 | 19.20% |
| Missing / incomplete H | 10,398 | 20.80% |
| Any preprocessing needed | 18,428 | 36.86% |

## Files

| File | Description |
|------|-------------|
| `csd_statistics.py` | Main sampling + analysis script (requires CSD Python API) |
| `plot_csd_results.py` | Figure S1 generation (requires `results/csd_statistics_raw.json`) |
| `results/csd_statistics_summary.json` | Aggregated statistics (counts + percentages) |

**Not included:** `csd_statistics_raw.json` (~4 MB, 50,000 per-entry records) — can be
regenerated by running `csd_statistics.py` with an active CSD API installation.

## Reproduction

```bash
# Run analysis (requires CSD Python API environment)
# On Windows with CCDC default installation:
cmd /c "D:\CCDC\ccdc-software\csd-python-api\miniconda\Scripts\activate.bat && python csd_statistics.py"

# Generate figure (standard Python with matplotlib)
python plot_csd_results.py
```
