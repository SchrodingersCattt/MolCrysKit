# Paper Reproducibility Data

This directory contains benchmark scripts and processed results for the paper:

> **MolCrysKit: A Topology-Aware Toolkit for Bridging Experimental Molecular-Crystal
> Structures and Simulation-Ready Modeling**
>
> Ming-Yu Guo and Wei-Xiong Zhang, *J. Chem. Inf. Model.*, 2026

## Contents

| Directory | Paper Figure | Description |
|---|---|---|
| `csd_statistics/` | Figure S1 | CSD sampling analysis — prevalence of disorder and missing H |
| `qm9_benchmark/` | Figures S2–S3, Table S1 | Hybridization inference validation on QM9 |
| `slab_benchmark/` | Figure 4a2 | Surface slab generation timing benchmarks |

## Quick Start

See each subdirectory's `README.md` for detailed instructions and dependencies.

### CSD Statistics (Figure S1)

Requires the [CSD Python API](https://www.ccdc.cam.ac.uk/solutions/software/csd-python-api/)
(not open-source; academic license required). The pre-computed summary results
are provided in `csd_statistics/results/`.

```bash
# Re-run analysis (requires CSD Python API)
python paper/csd_statistics/csd_statistics.py

# Re-generate figure from existing results
python paper/csd_statistics/plot_csd_results.py
```

### QM9 Benchmark (Figures S2–S3)

Requires the QM9 DFT xyz archive from
[figshare](https://figshare.com/collections/Quantum_chemistry_structures_and_properties_of_134_kilo_molecules/978904).
Download `dsgdb9nsd.xyz.tar.bz2` (~82 MB) and place it in `qm9_benchmark/data/`.

```bash
# Re-run validation
python paper/qm9_benchmark/validate_hybridization_qm9_real.py --n_samples 1000 --seed 42

# Re-generate figures from existing results
python paper/qm9_benchmark/plot_qm9_analysis.py
```

### Slab Benchmark (Figure 4a2)

```bash
# Re-run benchmark (may take several hours)
python paper/slab_benchmark/multi_system_benchmark.py

# Re-generate figure from existing results
python paper/slab_benchmark/plot_benchmark.py
```

## Notes

- The CSD raw per-entry data (`csd_statistics_raw.json`, ~4 MB) is not included
  in this repository due to size. It can be regenerated by running the
  `csd_statistics.py` script with an active CSD Python API installation.
- QM9 raw molecular data is public domain but large (~82 MB compressed); download
  instructions are in the QM9 benchmark README.
- All benchmark timing data was collected on a 4-core Intel Xeon Platinum 8163
  CPU @ 2.50 GHz (Bohrium Cloud Platform).
